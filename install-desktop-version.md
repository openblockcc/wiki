## 安装软件

1. 双击软件安装包，完成安装。
2. **首次安装后**，请在打开软件后点击右上角菜单安装驱动，以安装主板的驱动程序。

![安装驱动.png](/general-hardware-guidelines/boards/raspberry-pi-pico/安装驱动.png)

3. 等待驱动安装完成后就可以开始使用啦~。
